OVERVIEW
───────────────────────────────────────────────────────────
Text Editor Plus (SmartPad) is a powerful, modern text editor 
with syntax highlighting and auto-formatting capabilities.
Built with Python and PyQt5 for cross-platform compatibility.

FEATURES
───────────────────────────────────────────────────────────
✓ Syntax Highlighting for multiple languages:
  - XML/HTML, JSON, Python, CSS, JavaScript, and more

✓ Auto-Formatting:
  - XML/HTML formatting with proper indentation
  - JSON pretty printing
  - Python code formatting
  - CSS and JavaScript formatting

✓ Theme Support:
  - Dark Theme (default)
  - Light Theme
  - Smooth theme switching

✓ Advanced Editor Features:
  - Line numbers
  - Word wrap toggle
  - Find and replace with match counter
  - Undo/Redo support
  - Customizable font size (8-72pt)
  - Status bar with file information

✓ High DPI Support:
  - Perfect scaling on all screen sizes
  - Supports 100%, 125%, 150% DPI scaling

QUICK START
───────────────────────────────────────────────────────────
• Double-click TextEditorPlus.exe to launch the application
• Use Ctrl+O to open files
• Use Ctrl+S to save files
• Use Ctrl+Shift+F for auto-formatting
• Switch themes from Settings → Theme menu
• Press F1 for help

KEYBOARD SHORTCUTS
───────────────────────────────────────────────────────────
Ctrl+N          - New File
Ctrl+O          - Open File
Ctrl+S          - Save
Ctrl+Shift+S    - Save As
Ctrl+F          - Find
F3              - Find Next
Ctrl+Shift+F    - Auto-Format
Ctrl+Z          - Undo
Ctrl+Y          - Redo
Ctrl+X          - Cut
Ctrl+C          - Copy
Ctrl+V          - Paste
Ctrl+A          - Select All
F1              - Help
Ctrl+Q          - Exit
